Shared Shop - 精霊ショップ

Spring Boot と Oracle Database で構築した会員制 EC サイトを、森の精霊が営む小さな商店のような世界観で紹介します。

画面を見る 機能を読む GitHub

実装機能

基本的な EC フローに加えて、レビュー、お気に入り、クーポンガチャ、ログイン履歴など、購入前後の体験を支える機能を追加しました。

商品閲覧商品検索・カテゴリ検索・詳細表示
購入フロー買い物かご、注文確認、注文履歴
レビュー購入済み商品の評価とコメント投稿
お気に入り会員ごとの商品保存と一覧表示
クーポンマイクーポン、注文時利用、日次ガチャ
管理機能商品・カテゴリ・会員・注文管理
セキュリティ権限制御、アカウントロック、ログイン履歴
配送体験配送希望日と複数お届け先対応

クーポンガチャ

1日1回、午前7時に切り替わるクーポンガチャ。はずれ、5%OFF、10%OFF、15%OFF の結果を、注文時の割引につなげます。

日次制限同時実行対策注文時再検証

UIリニューアル

森林風の背景と装飾素材 絵本のような枠・リボン・木製ボタン 長い一覧はページングで整理
